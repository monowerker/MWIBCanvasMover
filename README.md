MWIBCanvasMover
===============

Xcode 5 Plug-In to move around the IB canvas with a third-party mouse more easily. Hold down spacebar, click-hold and drag to move around.

#### Installation

1. Clone Repo
2. Build in Xcode
3. Restart Xcode
